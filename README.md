## 🚀 About The Project

**JeevanSetu** (Bridge of Life) is a production-grade full-stack web application designed to solve India's organ donation crisis. JeevanSetu bridges that gap with , real-time notifications, and a seamless modern experience.

> 💡 One donor can save up to 8 lives — JeevanSetu makes that connection instant.

---

## ✨ Key Features

### 🫀 Donor Side
- Register and pledge organs (kidney, liver, cornea, heart, lungs, pancreas)
- Upload medical reports via Cloudinary
- Auto-generated digital donor card (PDF)
- Real-time notifications when matched
- In-app chat with matched recipient
- Track verification and match status

### 🏥 Recipient Side
- Post urgent organ requirements with urgency levels
- SOS emergency broadcast to all compatible donors
- Smart search with organ + blood group + city filters
- Real-time match notifications
- In-app chat with matched donor
- Track transplant journey step by step

### 🏨 Hospital Side
- Verify donor medical profiles
- Confirm donor-recipient matches
- Manage transplant status updates
- View pending verifications dashboard
